# Google Cloud Function Typescript Setup

## Available Scripts

### Test function

Will open a local server in ```http://localhost:8080```

```bash
yarn start
```

### Compile function

will creat an index.js file in ts-build directory for being used by the google cloud service

```bash
yarn build
```
